## Short description

## Long description

NeuFin Behavioral Signals is a private-beta MCP server for AI-augmented
wealth advisory. It exposes read-only tools that help advisor AI systems
surface client attention risk, churn probability, suitability drift, and
advisor-review context before workflows or agents take action.
